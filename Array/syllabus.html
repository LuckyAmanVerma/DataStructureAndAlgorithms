1. Prefix Sum / Sliding Window

Concept: Optimize subarray calculations (sum, average, etc.)
Problems:
  • Subarray Sum Equals K
  • Maximum Average Subarray
  • Minimum Size Subarray Sum
  • Count Number of Nice Subarrays
Tips:
  • Sliding window: use when subarray size is fixed
  • Prefix sum: for variable size & constant time queries
Practice: Dry-run with prefix sum arrays

2. Two Pointer Technique

Concept: Traverse from ends or maintain two moving indices
Problems:
  • 2Sum II (Sorted Input)
  • Container With Most Water
  • 3Sum / 4Sum
  • Move Zeroes / Remove Duplicates
Tips:
  • Use on sorted or "movable logic" arrays
  • Can reduce O(nÂ²) to O(n)
Practice: Visualize with two fingers moving

3. Binary Search on Array or Answer

Concept: Not just on values, but on search space
Problems:
  • Median of Two Sorted Arrays
  • Search Insert Position
  • Koko Eating Bananas
  • Aggressive Cows (Binary Search on Answer)
Tips:
  • Identify monotonicity to apply binary search
  • Focus on upper/lower bound tricks
Practice: Build template: low + high, mid, check(mid)

4. Greedy Logic in Arrays

Concept: Make local decisions to reach global optima
Problems:
  • Jump Game / Jump Game II
  • Can Place Flowers
  • Gas Station
Tips:
  • Track reachable indices or conditions
  • Dry-run each step and prove local â†’ global
Practice: Compare Greedy vs DP approaches

5. Hashing & Frequency Maps

Concept: Track counts, presence, or fast lookup
Problems:
  • Two Sum
  • Longest Consecutive Sequence
  • Majority Element
  • Group Anagrams
Tips:
  • Use unordered_map, set smartly
  • Think in terms of counts, not just values
Practice: Edge cases: duplicates, negatives, etc.

Concept: Find contiguous maximum subarray or max sum
Problems:
  • Maximum Subarray
  • Maximum Product Subarray
  • Circular Maximum Subarray
Tips:
  • For maximum: use max(curr, curr + num)
  • For circular: total sum - min subarray
Practice: Dry-run to build pattern memory

7. Monotonic Stack

Concept: For future-dependent questions â€” next greater/smaller
Problems:
  • Daily Temperatures
  • Next Greater Element II
  • Largest Rectangle in Histogram
Tips:
  • Stack maintains monotonicity (increasing/decreasing)
  • Often combined with reverse iteration
Practice: Simulate stack on paper

8. Divide & Conquer in Arrays

Concept: Break arrays into subarrays + combine results
Problems:
  • Merge Sort (base template)
  • Count Inversions
  • Maximum of Minimum in Every Window Size
Tips:
  • Use recursion or tree logic (segment trees)
  • Use merge steps to track cross-subarray info
Practice: Track indices while merging

9. 2D Arrays & Matrix Tricks

Concept: Think flattened, spiral, or matrix transformations
Problems:
  • Spiral Matrix
  • Search a 2D Matrix
  • Set Matrix Zeroes
  • Rotate Matrix
Tips:
  • Treat 2D indices as i/j, think in layers/levels
  • Flatten for binary search
Practice: Practice rotations visually

10. Bit Manipulation in Arrays

Concept: Use XOR or bit counting for unique elements
Problems:
  • Single Number / Single Number II
  • Missing Number
  • Sum of Two Integers
  • Bitwise AND of Range
Tips:
  • XOR of same numbers = 0 (cancels out)
  • Use 32-bit loops to count bit frequency
Practice: Convert to binary & dry-run bit masks
