Pehla Question: Subarray Sum Exactly Equal to k = 3

1. Prefix‐Sum Array ps Banana

Given:

nums = [1, 2, 3, −1, 2]

Calculate prefix sums:

ps[0] = 1
ps[1] = 1 + 2 = 3
ps[2] = 3 + 3 = 6
ps[3] = 6 + (−1) = 5
ps[4] = 5 + 2 = 7

ps = [1, 3, 6, 5, 7]

Initial: count = 0, m = {}

2. Iteration‐wise Trace

j = 0
- ps[0] = 1
- ps[0] == k ? 1 == 3 → no → count = 0
- target = 1 – 3 = −2
- m before = {} → m[−2] nahi mila
- m[1]++ → m = {1: 1}

j = 1
- ps[1] = 3
- ps[1] == k ? 3 == 3 → yes → count++ → count = 1
- target = 3 – 3 = 0
- m[0] nahi mila
- m[3]++ → m = {1:1, 3:1}

j = 2
- ps[2] = 6
- target = 6 – 3 = 3
- m[3] = 1 → count += 1 → count = 2
- m[6]++ → m = {1:1, 3:1, 6:1}

j = 3
- ps[3] = 5
- target = 5 – 3 = 2
- m[2] nahi mila → count = 2
- m[5]++ → m = {1:1,3:1,6:1,5:1}

j = 4
- ps[4] = 7
- target = 7 – 3 = 4
- m[4] nahi mila → count = 2
- m[7]++ → m = {1:1,3:1,6:1,5:1,7:1}

3. Final Result

Total subarrays with sum == 3 ⇒ 2
